# PriCloud

### setup

- clone the repo
- create a file in `./app/js/` called `sitesettings.js`:
```
var SITE=test/
var SITENAME="Test";
var DBSERVER="https://www.priocloud.com/testrest";
var USERSERVER="https://www.priocloud.com/testuser";
```
- run `npm install`

### development

- run `npm start`

### production 

- run `npm run build:production`
